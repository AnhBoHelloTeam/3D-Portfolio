# Automation Projects Images

This folder contains workflow diagrams, screenshots, and demo images for automation projects (n8n workflows).

## Required Images

Place the following images in this folder (matching the names in `src/constants/index.js`):

1. **crm-autolead-demo.png** - CRM Auto-Lead Capture workflow diagram
2. **sales-dashboard-demo.png** - Auto Daily Report & Alert workflow
3. **email-invoice-demo.png** - Email Invoice Bot workflow
4. **social-posting-demo.png** - Social Media Auto-Posting workflow
5. **inventory-sync-demo.png** - Inventory Sync & Low Stock Alert workflow
6. **ticket-router-demo.png** - Customer Support Ticket Router workflow
7. **backup-archive-demo.png** - Automated Data Backup & Archive workflow
8. **order-fulfillment-demo.png** - E-commerce Order Fulfillment Workflow

## Image Recommendations

- **Format**: PNG or JPG
- **Recommended size**: 900px - 1200px width
- **Content suggestions**:
  - Screenshots from n8n workflow editor
  - Flow diagrams from Draw.io, Lucidchart, or Miro
  - Architecture diagrams showing data flow
  - Before/after comparison screenshots

## Notes

- Images will be automatically served from `/automation/[filename]` in production
- Ensure images are optimized for web (compressed) to reduce load time
- If an image is missing, the UI will show a placeholder or fallback
